Kit contents

docs/brand/

├── atlas_mark_light.svg · combined (light)
├── atlas_sphere.svg · sphere standalone (dark)
├── atlas_sphere_light.svg · sphere standalone (light)

├── erisml_apple_light.svg · apple standalone (light)
├── README.md · usage guide, colors, do/don't rules
├── export.py · PNG + favicon.ico generator
├── index.html · this page
└── .gitignore · keeps exports/ out of git

Generating exports. Run pip install cairosvg pillow && python docs/brand/export.py to render the full PNG size matrix and favicon.ico under exports/. Commit individual export files only when needed in specific locations; keep the rest as build artifacts.

Usage rules. See README.md. Short version: don't scale the tether independently, don't recolor the apple, don't change the sphere/apple size ratio, use the right light/dark variant for the surface.
