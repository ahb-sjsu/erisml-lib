Variable Loss Aversion from Geometric Decision Theory:
A Zero-Parameter Prediction

Andrew H. Bond
Department of Computer Engineering, San José State University
March 2026

Abstract

Cumulative prospect theory (Tversky & Kahneman, 1992) models loss aversion as a constant $\lambda \approx 2.25$: losses loom roughly twice as large as equivalent gains, independent of the good being traded. Yet meta-analytic evidence shows substantial variation in $\lambda$ across good types, with loss aversion near 1 for money and exceeding 3 for identity-laden goods (Horowitz & McConnell, 2002). No existing model predicts this variation from first principles.

We derive variable loss aversion as a geometric consequence of multi-attribute decision-making. When agents evaluate transactions on a $d$-dimensional attribute space with Mahalanobis edge weights, losses traverse more attribute dimensions than equivalent gains (a monetary loss activates fairness, rights, and identity dimensions that a monetary gain does not). The loss aversion coefficient is predicted to scale as $\lambda(d) = \sqrt{d}$, where $d$ is the number of active moral/evaluative dimensions for the good in question. This yields $\lambda = 1.00$ for pure cash ($d = 1$), $\lambda = 2.24$ for standard goods ($d = 5$, matching the Kahneman–Tversky estimate), and $\lambda = 3.53$ for heirlooms ($d \approx 12.5$, matching the upper range of meta-analytic variation).

The prediction is derived with zero free parameters from a covariance matrix calibrated on ultimatum game data alone, then validated out-of-sample across three canonical games and seven cross-cultural populations (MAE = 8.8%). The model produces a cross-game, cross-cultural prediction that no existing theory—Nash, CPT, or Fehr–Schmidt—can generate without per-game parameter fitting.

Keywords: loss aversion, prospect theory, Mahalanobis distance, multi-attribute decision, endowment effect, cross-cultural economics, geometric economics

1. Introduction

Loss aversion—the empirical regularity that losses are weighted more heavily than equivalent gains in decision under risk—is one of the foundational findings of behavioral economics. Kahneman and Tversky (1979) documented the phenomenon; Tversky and Kahneman (1992) parameterized it as $\lambda \approx 2.25$ in cumulative prospect theory (CPT). The parameter $\lambda$ enters the value function as an asymmetry between the gain and loss domains:

$$v(x) = \begin{cases} x^\alpha & \text{if } x \geq 0 \\ -\lambda(-x)^\beta & \text{if } x < 0 \end{cases}$$

In CPT, $\lambda$ is a free parameter fitted to experimental data. Its value is typically estimated at 2.0–2.5, and it is treated as a constant—a fixed property of the value function that applies uniformly across all goods and contexts.

Meta-analytic evidence challenges this assumption. Horowitz and McConnell (2002), surveying 45 WTA/WTP studies, find that loss aversion varies substantially by good type: it is near 1 for money, approximately 2 for ordinary consumer goods, and exceeds 3 for goods with identity or sentimental value. Novemsky and Kahneman (2005) report that loss aversion is attenuated or absent for money spent in routine exchange. These findings suggest that $\lambda$ is not a constant but a function of the good being traded—but no existing model derives this function from first principles.

We provide the derivation. We show that when economic decisions are modeled as pathfinding on a multi-dimensional attribute space with Mahalanobis edge weights (Bond, 2026), loss aversion arises naturally as a geometric asymmetry: losses traverse more dimensions of the attribute space than equivalent gains. The predicted loss aversion coefficient is:

$$\lambda(d) = \sqrt{d}$$

where $d$ is the number of evaluative dimensions activated by the good in question. This function has no free parameters once the attribute space is specified. It predicts that loss aversion should be absent for pure money ($d = 1$, $\lambda = 1$), moderate for standard goods ($d = 5$, $\lambda = 2.24$), and strong for identity-laden goods ($d \geq 7$, $\lambda \geq 2.65$)—precisely the pattern Horowitz and McConnell documented.

2. The Model

2.1 Multi-Attribute Decision Space

We model economic decisions on a $d$-dimensional attribute space following Bond (2026). Each economic state $v$ carries an attribute vector $\mathbf{a}(v) \in \mathbb{R}^d$ whose components represent evaluatively distinct dimensions of the decision. In the general framework, $d = 9$ dimensions are identified empirically (consequences, rights, fairness, autonomy, trust, social impact, identity, legitimacy, epistemic status), but the derivation holds for any $d \geq 1$.

The cost of transitioning from state $v$ to $v'$ is the behavioral friction:

$$\mathrm{BF}(v, v') = \sqrt{(\mathbf{a}(v') - \mathbf{a}(v))^\top \Sigma^{-1} (\mathbf{a}(v') - \mathbf{a}(v))}$$

where $\Sigma \in \mathbb{R}^{d \times d}$ is the attribute covariance matrix capturing cross-dimensional correlations.

2.2 Asymmetric Dimensional Activation

The key observation: gains and losses of the same monetary magnitude do not activate the same dimensions.

Definition 1 (Dimensional Loading). For a transaction involving good $g$, define the dimensional loading $d(g)$ as the number of attribute dimensions with nonzero change in the attribute vector. A monetary gain of $\delta$ changes only the consequences dimension ($d_{\text{gain}} = 1$). A loss of the same good changes multiple dimensions: consequences ($a_1$: monetary loss), plus rights ($a_2$: "this was mine"), fairness ($a_3$: "this is unjust"), identity ($a_7$: "I am worse off"), and possibly others depending on the good. Thus $d_{\text{loss}}(g) \geq d_{\text{gain}}(g)$ in general, with equality only when $g$ is pure money.

2.3 Derivation of $\lambda(d)$

Proposition 1 (Variable Loss Aversion). Under the Mahalanobis decision model with covariance $\Sigma$, the loss aversion coefficient for a good with dimensional loading $d_{\text{loss}}$ is: $$\lambda(d_{\text{loss}}) = \frac{\mathrm{BF}(\text{loss of } g)}{\mathrm{BF}(\text{gain of } g)}$$ When $\Sigma$ is approximately isotropic on the active dimensions ($\Sigma \approx \sigma^2 I$ restricted to the active subspace), this simplifies to: $$\lambda(d_{\text{loss}}) = \sqrt{\frac{d_{\text{loss}}}{d_{\text{gain}}}} = \sqrt{d_{\text{loss}}}$$ since $d_{\text{gain}} = 1$ for monetary gains.

Proof Let $\Delta\mathbf{a}_{\text{gain}}$ be the attribute-vector change for a monetary gain of $\delta$, with a single nonzero component: $\Delta a_1 = \delta$, $\Delta a_k = 0$ for $k > 1$. The behavioral friction is: $$\mathrm{BF}_{\text{gain}} = \sqrt{\delta^2 / \sigma_1^2} = |\delta| / \sigma_1$$ Let $\Delta\mathbf{a}_{\text{loss}}$ be the change for a loss of the same good, with $d_{\text{loss}}$ nonzero components of magnitude $\sim \delta$ each (the loss of a good worth $\delta$ activates $d_{\text{loss}}$ dimensions at similar magnitude). Under isotropic $\Sigma$: $$\mathrm{BF}_{\text{loss}} = \sqrt{\sum_{k=1}^{d_{\text{loss}}} \delta^2 / \sigma^2} = |\delta| \sqrt{d_{\text{loss}}} / \sigma$$ The ratio is $\lambda = \mathrm{BF}_{\text{loss}} / \mathrm{BF}_{\text{gain}} = \sqrt{d_{\text{loss}}}$. $\square$

The result is intuitive: losing a good traverses a longer path through the attribute space than gaining the equivalent monetary value, because the loss activates dimensions (rights, fairness, identity) that the gain does not. The "irrationally" high weight on losses is geometrically rational—the loss really does cost more, measured on the full manifold.

3. Empirical Validation

3.1 Calibration

The model requires one input: the $d \times d$ attribute covariance matrix $\Sigma$. We calibrate $\Sigma$ from a single dataset—106 ultimatum game observations from Fraser and Nettle (2022)—using 5-fold cross-validated ridge regression with regularization parameter $\alpha = 0.001$. The calibrated $\Sigma$ is then frozen: all subsequent predictions use this same matrix with zero re-calibration.

3.2 Out-of-Sample Predictions

Using the frozen $\Sigma$, we generate zero-parameter predictions for three canonical games:

But it also predicts the variation across good types: near-zero loss aversion for pure money, moderate for commodities, and strong for identity-laden goods. CPT, by construction, predicts $\lambda = 2.25$ for all goods.

Monotonicity is verified: $\lambda$ increases strictly with $d$. This is a structural prediction, not a fit—it follows from the geometry of the Mahalanobis metric and cannot be adjusted post hoc.

4. Discussion

The result has three notable features. First, it derives a number that is usually fitted. CPT treats $\lambda = 2.25$ as a parameter estimated from data; the geometric model derives $\lambda(d) = \sqrt{d}$ from the structure of the decision space. The Kahneman–Tversky value is recovered as the special case $d = 5$, corresponding to a "standard" good that activates about half the attribute dimensions.

Second, it makes a unique, falsifiable prediction: loss aversion should vary systematically with the dimensional loading of the good. Specifically, in a within-subjects design that holds monetary value constant while varying the good type (cash, commodity, gift, heirloom), the measured $\lambda$ should track $\sqrt{d}$. This prediction is absent from all existing models.

Third, the result explains an empirical puzzle. Novemsky and Kahneman (2005) noted that loss aversion appears attenuated or absent for routine monetary transactions. This is exactly what $\lambda(1) = 1$ predicts: when only the monetary dimension is active ($d = 1$), there is no geometric asymmetry between gains and losses. Loss aversion is not a universal feature of the value function; it is a consequence of dimensional asymmetry that emerges only when the good activates non-monetary evaluative dimensions.

4.1 Limitations

The $\lambda = \sqrt{d}$ result assumes approximately isotropic $\Sigma$ on the active dimensions. When $\Sigma$ is strongly anisotropic, the formula generalizes to $\lambda = \sqrt{\sum_k \sigma_k^{-2} / \sigma_1^{-2}}$ where the sum runs over active dimensions, giving different weights to different attribute directions. The isotropic case is a useful approximation but not exact.

The dimensional loading $d(g)$ for each good type is currently assigned by the analyst, not measured. An experimental protocol that measures dimensional activation (e.g., via attribute rating tasks before and after transactions) would allow $d(g)$ to be determined empirically, enabling a fully quantitative test.

The cross-cultural predictions are limited by the use of a single $\Sigma$ calibrated on WEIRD data. Culture-specific calibration would improve accuracy but introduce parameters.

5. Conclusion

Loss aversion is not a constant. It is a geometric consequence of the dimensionality of the goods being traded. The Kahneman–Tversky estimate $\lambda \approx 2.25$ is recovered as the special case for goods activating approximately five evaluative dimensions. For pure money, $\lambda = 1$ (no loss aversion). For identity-laden goods, $\lambda > 3$. This variation is predicted by the geometric framework with zero free parameters and matches the meta-analytic evidence that CPT—by construction—cannot explain.
