Moral Heuristics as Admissible Search:
A Computational Theory of Dual-Process Decision-Making

Andrew H. Bond
Department of Computer Engineering, San José State University
March 2026

Abstract

We propose that Kahneman's dual-process architecture—System 1 (fast, automatic) and System 2 (slow, deliberate)—is not merely a psychological description but an instance of a well-characterized computational architecture: A* search with a heuristic evaluation function. System 2 computes the accumulated cost $g(n)$ of the path so far (deliberate cost-benefit analysis); System 1 provides the heuristic estimate $h(n)$ of the remaining cost to the goal (moral intuitions, social norms, cultural rules). The agent's decision is the path that minimizes $f(n) = g(n) + h(n)$.

We prove that moral and cultural heuristics, under mild conditions, are admissible—they never overestimate the true cost-to-go on the full decision manifold—and therefore A* search with System 1 as the heuristic is guaranteed to find the optimal path. The "biases" identified by behavioral economics are classified into four categories: (1) strictly admissible heuristics (not biases—optimal on the full manifold), (2) $\varepsilon$-admissible heuristics (bounded suboptimality, the evolutionary sweet spot), (3) inadmissible heuristics (genuine cognitive errors), and (4) gauge-variant heuristics (framing effects). Only categories 3 and 4 warrant the label "bias." Categories 1 and 2 are features of an efficient search architecture.

Keywords: dual-process theory, A* search, admissible heuristic, bounded rationality, moral heuristics, System 1, System 2, behavioral economics, decision-making

1. Introduction

Kahneman's (2011) dual-process theory partitions cognition into two systems: System 1 operates automatically and quickly, with little effort and no sense of voluntary control; System 2 allocates attention to effortful mental activities, including complex computations. This architecture has become the default framework for understanding judgment and decision-making across psychology, behavioral economics, and cognitive science.

Yet dual-process theory lacks a formal computational account. What computation does System 1 perform? What computation does System 2 perform? What is the formal relationship between them? Why does their interaction produce the specific pattern of "biases" documented by behavioral economics? Without a computational theory, dual-process theory remains a description—it names the phenomena but does not explain them.

We provide the missing computational theory. We show that the dual-process architecture is an instance of A* search (Hart, Nilsson, & Raphael, 1968)—the most studied algorithm in artificial intelligence for optimal pathfinding. The mapping is not metaphorical. It is exact:

The key result: if System 1's moral heuristics are admissible (never overestimate the true cost-to-go), then the dual-process system finds the optimal action. The "biases" are not errors—they are the signature of an efficient search algorithm that trades exhaustive computation for heuristic-guided pruning.

2. The Formal Mapping

2.1 The Decision Space

The agent's decision problem is defined on a weighted graph $\mathcal{C} = (V, E, w)$ where vertices $V$ are economic/social states, edges $E$ are available actions, and $w : E \to \mathbb{R}_{\geq 0}$ assigns multi-dimensional costs (behavioral friction) to actions. The agent seeks a minimum-cost path from the current state $v_0$ to a goal region $G \subset V$.

2.2 System 2 as $g(n)$

Definition 1 (System 2 cost). For a partial path $\pi = (v_0, v_1, \ldots, v_k)$, the System 2 evaluation is: $$g(v_k) = \sum_{t=0}^{k-1} w(v_t, v_{t+1})$$ This is the accumulated behavioral friction along the path so far—the explicit, deliberate cost-benefit calculation.

2.3 System 1 as $h(n)$

Definition 2 (System 1 heuristic). The System 1 evaluation at state $v$ is: $$h(v) = \sum_{k \in \mathcal{B}(v)} \beta_k$$ where $\mathcal{B}(v)$ is the set of moral/social boundaries that must be crossed on any path from $v$ to the goal, and $\beta_k$ is the estimated boundary penalty. This is the fast, automatic assessment: "how morally/socially costly is it to get from here to where I want to be?"

3. Main Results

3.1 The Admissibility Theorem

Theorem 1 (Heuristic Admissibility). Let $h^*(v)$ denote the true minimum remaining cost from $v$ to the goal on the full decision manifold $\mathcal{C}$. If the boundary penalties $\beta_k$ are lower bounds on the true cost of crossing boundary $k$ (i.e., $\beta_k \leq \beta_k^*$ where $\beta_k^*$ includes legal, social, psychological, and reputational costs across all attribute dimensions), then $h(v) \leq h^*(v)$ for all $v$, and A* search with heuristic $h$ finds the optimal path on $\mathcal{C}$.

Proof The true remaining cost $h^*(v)$ includes the actual boundary penalties $\beta_k^*$ plus the Mahalanobis distance to the goal along the optimal remaining path. Since $\beta_k \leq \beta_k^*$ and the Mahalanobis component is non-negative, $h(v) = \sum_k \beta_k \leq \sum_k \beta_k^* \leq h^*(v)$. By the admissibility theorem for A* (Hart et al., 1968), the search finds the optimal path. $\square$

Corollary 1 (Moral Rules Are Not Biases). If moral heuristics are admissible, then following them produces optimal behavior on the full decision manifold. The behavior appears suboptimal only when evaluated on a scalar projection (e.g., monetary payoff alone). The "bias" is in the evaluation metric, not in the behavior.

3.2 The Admissibility Hierarchy

Figure 2. The admissibility hierarchy. Behavioral economics has classified all four categories as "biases." Categories 1–2 are architectural features of an efficient search algorithm; only categories 3–4 are genuine cognitive errors.

Definition 3 ($\varepsilon$-Admissibility). A heuristic $h$ is $\varepsilon$-admissible if $h(v) \leq (1 + \varepsilon) h^*(v)$ for all $v$. Weighted A* with an $\varepsilon$-admissible heuristic finds a path whose cost is at most $(1 + \varepsilon)$ times the optimal cost, while exploring exponentially fewer states than exact search (Pohl, 1970).

Theorem 2 (Evolutionary Optimality of $\varepsilon$-Admissibility). Natural selection favors $\varepsilon$-admissible moral heuristics with small $\varepsilon > 0$ over strictly admissible heuristics ($\varepsilon = 0$) because:

  1. Weighted A* with $\varepsilon > 0$ explores $O(n^{1/(1+\varepsilon)})$ states vs. $O(n)$ for exact A*, yielding faster decisions.
  2. Heuristics biased toward overestimation of boundary costs ($h > h^*$ for dangerous boundaries) avoid catastrophic paths at the cost of small opportunity losses.
  3. An organism with $h < h^*$ (underestimates boundary costs) occasionally crosses catastrophic boundaries and is eliminated from the gene pool.
Evolution therefore produces System 1 heuristics that are conservative: slightly overestimating sacred-boundary costs ($\varepsilon > 0$) while maintaining near-optimality.

3.3 The Jean Valjean Problem

Remark 1 (When Systems Conflict). A natural objection: if "never steal" assigns $h = \infty$ but the true cost of not stealing (starvation) also approaches infinity, then Systems 1 and 2 conflict. The framework predicts what is observed empirically: (1) the decision is experienced as agonizing moral conflict ($g$ and $h$ give contradictory signals); (2) the agent deliberates slowly (System 2 overrides System 1's automatic pruning); (3) society treats the transgression with nuance (the community recognizes the manifold distortion). This is not a failure of the model—it is the model's prediction for extreme boundary cases.

4. Reclassifying Behavioral "Biases"

We reclassify the major findings of behavioral economics within the admissibility hierarchy:

PhenomenonCategoryMechanism
Theft aversion1 (admissible)$h = \infty$ for theft boundary; true cost is indeed effectively infinite
Fairness in ultimatum game1–2$h$ includes social-norm penalty; admissible on full manifold
Tipping, reciprocity2 ($\varepsilon$-admissible)Small overestimate of social cost; near-optimal
Loss aversion2Losses traverse moral dimensions (rights, fairness) that gains do not; $h$ is larger for loss paths—this is correct, not biased
Phobias3 (inadmissible)$h \gg h^*$ for harmless stimuli; genuine miscalibration
Framing effects4 (gauge-variant)$h$ depends on description, not state; inconsistent function
Anchoring4$h$ contaminated by irrelevant initial value
Availability bias4$h$ driven by memory salience, not actual frequency

This reclassification has a sharp empirical prediction: Category 1–2 phenomena should be stable across cultures and contexts (they reflect manifold structure, not cognitive error), while Category 3–4 phenomena should be variable (they reflect individual miscalibration or informational manipulation). This prediction is consistent with the cross-cultural evidence: fairness norms and theft aversion are near-universal (Henrich et al., 2001), while framing effects and anchoring vary substantially across populations and experimental conditions (Kahneman, 2011).

5. Computational Complexity: Why Heuristics Are Necessary

Proposition 1 (Intractability of Exact Optimization). Finding the optimal path on a $d$-dimensional decision manifold with $n$ states requires evaluating $O(n^d)$ attribute-vector comparisons in the worst case. For realistic parameters ($n \sim 10^4$ available actions, $d = 9$ dimensions), exact optimization is intractable in the milliseconds available for real-time decision-making. A* with an admissible heuristic reduces this to $O(n \cdot b^{d_{\text{eff}}})$ where $b$ is the branching factor and $d_{\text{eff}} \ll d$ is the effective search depth after heuristic pruning.

This is why evolution produced System 1. Not because organisms are "irrational," but because exact optimization on a 9-dimensional manifold is computationally intractable. The heuristic-guided search trades global optimality for tractable near-optimality—precisely the trade-off that bounded rationality (Simon, 1955) identified but did not formalize.

6. Discussion

The A* mapping resolves a persistent tension in cognitive science. Dual-process theories describe the architecture; rational-agent models prescribe the objective; behavioral economics documents the gap between them. The A* framework unifies all three: the architecture is the rational agent, implemented as heuristic search rather than exhaustive optimization.

Several testable predictions follow: (1) Decisions that activate more moral dimensions ($d_{\text{eff}}$ increases) should take longer (more nodes expanded in A*). (2) Decisions near infinite-weight boundaries should be fast (System 1 prunes instantly) unless the alternative path is also high-cost (System 1/2 conflict, predicting slow agonized deliberation). (3) Expertise in a domain should improve heuristic calibration ($\varepsilon$ decreases), making experts' intuitions more optimal, not less—consistent with Gigerenzer's findings on fast-and-frugal heuristics in experts.

The framework also makes a normative claim: calling admissible heuristics "biases" is a category error. When the full attribute space is the correct optimization domain, System 1's moral intuitions are not obstacles to rationality—they are components of it.

7. Conclusion

We have shown that Kahneman's dual-process architecture has a precise computational identity: A* search with System 1 as the heuristic function and System 2 as the path-cost accumulator. Moral and cultural heuristics, under mild conditions, are admissible—guaranteeing optimal or near-optimal decisions on the full multi-dimensional decision manifold. The "biases" of behavioral economics decompose into architectural features (admissible and $\varepsilon$-admissible heuristics) and genuine cognitive errors (inadmissible and gauge-variant heuristics). This decomposition is falsifiable, culturally testable, and computationally precise.
